# Mavis's Vault

A PARA-style second brain, mirrored in Obsidian.

## Sections

- 📥 [[00 - Inbox]] — quick capture
- 📅 [[01 - Daily Notes]] — daily journal
- 🎯 [[02 - Projects]] — active, time-bounded
- 🌱 [[03 - Areas]] — ongoing responsibilities
- 📚 [[04 - Resources]] — reference material
- 🗄️ [[05 - Archives]] — completed / inactive

## Conventions
- **Wikilinks** (`[[note name]]`) everywhere — they're how the graph builds
- **Numbered prefixes** keep folder order stable
- **README.md per folder** = the section hub
- **Process [[00 - Inbox]] weekly** — that's the whole point of PARA
