# Track G — Take-homes & Work Trials  (master plan §7.5)

**Charter:** several targets gate on async assignments. They're a **relative advantage for you** (infra-strong, more time than a 45-min live round) — *if* treated as a scored work sample, not a quick script. Most candidates lose on **polish, scope, and communication**, not raw ability. Budget is small + **reactive**: build the reusable scaffold now so a real one is execution, not panic.

**Who leans on it:** **Cohere (gating round, ~30% pass)**, OpenAI (work-trial), Anthropic (90-min, 4-level assessment), Harvey, Anysphere, Together. Cross-ref [[cohere-interview-guide]] and [[anthropic-interview-guide]] (AI-usage policy).

> **The 3 take-home types** (was `take-home-types.md`, now folded into [[per-company-notes]]): **(1) Timed assessment** (Anthropic, CodeSignal) → correctness + clean incremental design under time, prep = Tracks A/B · **(2) Open build** (OpenAI, Harvey, Anysphere, Together) → working software + scope judgment + tests + clear README · **(3) Analysis + presentation** (Cohere) → a *narrative*: framing → method → findings → recommendation → caveats, graded on communication.

## Override rule
A live take-home (esp. **Cohere**) outranks everything that week — pour **8–15 focused hrs** into it using the scaffold. See master plan §7.5 and the schedule "take-home override."
