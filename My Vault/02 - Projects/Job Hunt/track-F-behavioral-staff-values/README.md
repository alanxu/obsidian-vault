# Track F — Behavioral / Staff Narrative / Values  (master plan §7)

**Charter:** the #1 failure point at frontier labs. Staff level = stories that show **scope, technical leadership, influence without authority, judgment under ambiguity, multiplying others** — not "I coded a feature." Cheap to prep, decisive in outcome.

**Who leans on it:** all 12 — and it's the deciding round at Anthropic/OpenAI.

## To elaborate here
- [ ] `story-bank.md` — 6–8 STAR stories, each tagged with the staff signal + a 30-sec and 2-min version:
  1. Biggest technical impact  2. Cross-team influence / drove consensus  3. Disagreement/conflict
  4. Failure & lesson  5. Ambiguous problem  6. Tradeoff under constraint  7. Mentoring/multiplying  8. Incident/firefight
- [ ] `values-hooks-per-company.md` — one specific "why this company" line + values talking points (Anthropic safety/RSP, OpenAI mission, Google Googliness/GCA, Robinhood/Wealthsimple democratizing finance, Waymo/Waabi safety-critical, NVIDIA performance)
- [ ] `questions-to-ask.md` — 3–4 sharp questions per interviewer type
- [ ] `ai-safety-pov.md` — a genuine point of view (required for Anthropic; see [[anthropic-interview-guide]])

## Method
Lead with the result, then how. Quantify. Name the tradeoff you owned. Record yourself for the values round.
