# 📅 Daily Notes

One note per day. File naming: `YYYY-MM-DD.md`.

## How to use
- Open today's note (use the Daily Notes core plugin or the Templater plugin)
- Capture: top of mind, priorities, win of the day, gratitude, open loops
- Link people ([[name]]) and projects ([[02 - Projects/...]]) as you write
